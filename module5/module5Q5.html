Spaghetti Code Training Manual


Ethics:

Spaghetti code has ethical concerns related to security for the end user. Spaghetti code creates the increased potential for unforeseen code usage or manipulation. The Heartbleed vulnerability provides a strong example of the ethical dangers of spaghetti code. Spaghetti code is often made by hastily putting code together. The implementation of the Heartbeat extension into OpenSSL allowed for the retrial of any information in the memory buffer.
