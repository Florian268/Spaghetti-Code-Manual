Code Structure

The first and most important part of code structure is the Open-Close Rule. Every time you create an opening tag, such as '(', '[', or '{', you must remember to close that tag with it's partner tag ')', ']', or '}'. Depending on what language you write in, different objects will use different tags to determine functionality, so don't worry about jargon. Just focus on the structure. Every function call should have a set of parenthesis '()' directly after the call. If I have a function named "foo", I would call it by typing "foo(argument1, argument2, etc)". You access an array by typing "arrayName[i]" where i is the position of the element you want to access.
